# Text Analyzer - React App

A simple React app that analyzes text input and shows:

- Word count
- Letter count (excluding spaces)

Built with **React** and **Bootstrap**.  

## How to Run
npm install
npm run dev

Open http://localhost:5173 in your browser.

## Features
- Input text and click Analyze
- Shows results below the input
- Beginner-friendly React project
